# Blueprint - Thiết kế hệ thống

## 1. Đọc INTAKE
// turbo
Đọc `docs/INTAKE.md` để hiểu yêu cầu

## 2. Thiết kế
Xác định:
- **Tech Stack**: Frontend, Backend, Database, Hosting (+ lý do chọn)
- **Modules**: Liệt kê + mục đích từng module
- **User Flows**: Flow chính + các bước
- **Data Model**: Entities + relationships
- **Phases & Gates**: Phase 1 làm gì, checkpoints

## 3. Tạo BLUEPRINT.md
// turbo
Viết thiết kế vào `docs/BLUEPRINT.md`

## 4. Tạo DECISIONS.md
// turbo
Ghi lại quyết định tech stack:
```markdown
## ADR-001: Chọn [Tech]
**Ngày:** [Hôm nay] | **Trạng thái:** Proposed
**Lý do:** [Giải thích]
```
