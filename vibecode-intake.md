# Intake - Thu thập yêu cầu

## 1. Hỏi thông tin
1. **Mục tiêu**: Kết quả cuối cùng? 6-12 tháng nữa như thế nào?
2. **Đối tượng**: Ai dùng? Họ gặp vấn đề gì?
3. **Ràng buộc**: Timeline? Budget? Tech stack bắt buộc?
4. **Phạm vi**: Tính năng must-have? Để sau?
5. **Ưu tiên**: Điều gì quan trọng nhất Phase 1?

## 2. Tạo INTAKE.md
// turbo
```markdown
# INTAKE – [Tên dự án]
## 1. Mục tiêu: [...]
## 2. Đối tượng: [...]
## 3. Ràng buộc: [...]
## 4. Phạm vi: [...]
## 5. Yêu cầu kỹ thuật: [...]
## 6. Ưu tiên & thành công: [...]
```
